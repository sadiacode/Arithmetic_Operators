# Arithmetic Operators
## Description
This project is a simple python program that demonstrates the use of arithmetic operators. The program performs arithmetic operations on two numbers and displays the result.
## Operators Used
- Addition('+')
- Subtraction('-')
- Multiplication('*')
- Division('/')
- Floor Division('//')
- Modulus('%')
- Exponential('**')
## Technologies Used
- Pycharm
- Python 3
## How to run
1. Clone the repository.
2. Open the project in pycharm.
3. Run the 'main.py' file.
4. The program performs arithmetic operations on two numbers.
5. View the result of calculation.
## Sample Output
```Operators

10+3=13
10-3=7
10*3=30
10/3=3.33333333
10//3=3
10%3=1
10**3=1000
```
